Blender Add-on

Base Character Modeling

Fast part-based character blockout with curated templates and reference tools for Blender.

Install: download the add-on ZIP, then in Blender Preferences → Add-ons → Install...

How to install

Features

Reference Images

Load and align front/side/hand-top images with shared transform controls so your sculpt matches references.

Part Templates

Use shipped templates or create scene-local templates to quickly build parts that fit together.

Build Tools

Build a full base character or individual parts with status indicators for shipped/custom/missing templates.

Alignment Helpers

Landmark-based alignment checks and utility operators to keep scale and ground consistent.

Install & Usage

  1. Download the add-on ZIP from the repository release.
  2. In Blender: Edit → Preferences → Add-ons → Install..., select the ZIP.
  3. Enable the add-on, open the side-panel and follow the Reference / Template panels.
